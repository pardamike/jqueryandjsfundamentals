Section 2: Advanced Selection, Events, Listeners, and Inputs

Overview: Today we will cover more advanced selection, events, event listeners, as well as inputs. All of these things often relate to each other, so lets get started!

How can I select...?

Basic DOM selection in both Javascript and jQuery is pretty strightforward as we saw in the last section. However, we often find ourselves with much harder items to select, and sometimes we cannot change the existing HTML code to give an element a unique ID to select it by.

Let's take a look at some difficult selections...

The plugin generated grid

We have all worked with plugin generated grids at some point. If you are stuck with the wrong plugin, it can be very difficult to do any DOM manipulations and selections with a grid. Enterprise UI toolkits that are old are pretty notorious for this.
